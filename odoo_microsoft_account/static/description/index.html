Microsoft Azure SSO Integration

A Microsoft Azure SSO Integration module has been released by SerpentCS. You can sync your Microsoft Azure users seamlessly with Odoo using this module. We have got 2 more modules, which is Calendar Sync and Contact Sync. These modules will allow you to sync Office.com calendar events and contacts with Odoo calendar events and contacts.

Create Application

You Need to Enter Following Details

  • When we go to App registrations page we will be shown number of applications that are already created and needs to connect with Odoo.

Application Details:

    Inside application, user will be able to see application id and client credential. These credentials are required to be copied in Odoo setup to make connection between Microsoft account and Odoo appllication.

Url:

  • The URL of Odoo application is to be added in the URL details of application of Microsoft Azure portal.

Odoo Setup :

  • Inside Odoo first go to Settings --> Start developer mode --> User & Companies --> OAuth Proivders .

Provider Details:

  • Inside the Providers there will be number of provider details will be there. We need to go to “Microsoft Oauth2” provider details.

Provider Setup:

  • Add the detail taken from Microsoft Account application in Microsoft Oauth2 details.

Login Screen:

  • On login screen you will be able to look at button to “Login with Microsoft” click on that button and you will be redirected to microsoft login page.

Microsoft Login Page:

  • On Microsoft Login page login using your microsoft ID and you will be redirected to Odoo backend as per your user need.

Odoo Back End and User details:

  • A new user will be added with Microsoft account details and user will be able to login using microsoft id through Odoo.

Disable/Enable Odoo Debug Mode

This module allow users to enable and disable odoo debug option if they have access to 'Enable/Disable Debug Mode' group.

Funtionalities:
  • By Default No Users has access to debug mode options unless added to 'Enable/Disable Debug Mode' group.
  • Users without access to 'Enable/Disable Debug Mode' will be redirected back without activating/deactivating the debug mode.
  • Users with access to 'Enable/Disable Debug Mode' will be able to activate/deactivate the debug mode.
Step 1: Install module

To install the module, go to Apps and search for module name and install it.


Step 2: Add Group To user
  • Default No user has access to enable and disable debug mode
  • To Allow users, go to Settins/users/{select one user}/under 'other' category/Check Enable/Disable Debug Mode

Step 3: Testing
  • After Adding Access group to user, enable debug mode
  • And the debug icon will be visible.
